Exercices
Règles clés
📌 Accord du participe passé
Avec AVOIR : Accord avec le COD si placé avant.
Ex : Les lettres qu'il a écrites.
Avec ÊTRE : Accord avec le sujet toujours.
Ex : Elles sont arrivées.
📌 Pluriel des noms composés
Nom + Nom : Les deux prennent le pluriel.
Ex : des choux-fleurs → des choux-fleurs
Verbe + Nom : Seul le nom prend le pluriel.
Ex : des porte-clés
📌 Conjugaison — Subjonctif présent
Il faut que je fasse, que tu ailles, qu'il soit, que nous ayons.
Après : il faut que, bien que, pour que, quoique…
📌 Homophones courants
a/à · est/et/es · ou/où
ce/se · ces/ses/c'est/s'est
mais/mes/met/mets
Test : remplace par "avait" pour distinguer a/à.
